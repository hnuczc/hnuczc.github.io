Linux

Shell基础

Shell概述

Shell是命令行解释器，也是功能强大的编程语言

Linux支持的Shell列在文件/etc/shells里面，常用：bash，sh，csh

Shell脚本执行方式

  1. 输出命令 echo

    echo [选项] [输出内容]

    ​ -e：支持反斜线控制的字符转换

    控制字符 作用
    \\ 输出\本身
    \a 输出警告声
    \b 退格键（向左删除）
    \n 换行
    \t 制表符（Tab）
    \v 垂直制表符
    常用如上
    \0nnn 按照八进制ASCII表输出，其中0是数字0，nnn是三位八进制数
    \xhh 按照十六进制ASCII表输出，其中hh是两位十六进制数
    有趣的如下
    \e[1;31m 内容 \e[0m 输出颜色，30m=黑色，31m=红色，32m=绿色，33m=黄色，34m=蓝色，35m=洋红，36m=青色，37m=白色

Bash的基本功能

  1. 历史命令与命令补全

    1. history [选项] [历史命令保存文件]

    ​ -c 清空历史命令

    ​ -w 把缓存中的历史命令写入历史命令^保存文件中

  1. 历史命令默认会保存1000条，可以在环境变量配置文件/etc/profile中，修改环境变量HISTSIZE

  2. 历史命令的调用

  • 使用上、下箭头调用以前的历史命令
  • 使用”!n”重复执行第n条命令
  • “!!”重复执行上一条命令
  • “!字串”重复执行最后一条以该字串开头的命令

  1. 在输入命令或文件时，按Tab键会自动补全

    当有多个文件开头相同时，无法触发Tab补全功能，可以连按两下Tab键，输出开头相同的文件或命令

  1. 命令别名与常用快捷键

    1. 命令别名

      1. 设定命令别名 alias 别名=‘原命令’

      2. 查询命令别名 alias

      3. 命令执行时的顺序

        1. 执行用绝对路径或相对路径执行的命令
        2. 执行别名
        3. 执行Bash的内部命令
        4. 按照$PATH环境变量定义的目录查找顺序的找到的第一个命令

      4. 让别名永久生效

        用vi 打开/root/.bashrc ，在其中添加需要添加的别名

      5. 删除别名

        unilias 别名

    2. Bash常用快捷键

      快捷键 作用
      ctrl+C 强制终止当前命令
      ctrl+L 清屏，相当于clear
      ctrl+U 删除或剪切当行光标前的命令
      ctrl+K 删除或剪切当行光标后的命令
      ctrl+Y 粘贴ctrl+U或者ctrl+K剪切的内容
      ctrl+R 在历史命令中搜索。按下ctrl+R之后，会出现搜索界面。
      ctrl+D 退出当前终端
      ctrl+Z 暂停，放到后台

  1. 输入输出重定向

    1. 标准输入输出

      设备 文件描述符 类型
      键盘 0 标准输入
      显示器 1 标准输出
      显示器 2 标准错误输出

    2. 输出重定向

      类型 符号 作用
      标准输出重定向 命令 > 文件 以覆盖的方式，把命令正确输出输出到制定文件或设备中
      标准输出重定向 命令 >> 文件 以追加的方式，把命令正确输出输出到制定文件或设备中
      标准错误输出重定向 命令 2> 文件 以覆盖的方式，把命令错误输出输出到制定文件或设备中
      标准错误输出 命令 2>> 文件 以覆盖的方式，把命令错误输出输出到制定文件或设备中

      正确输出和错误输出同时保存

      符号 作用
      命令 > 文件 2>& 1 以覆盖的方式，把正确输出和错误输出都保存到同一个文件中
      命令 >> 文件 2>& 1 以追加的方式，把正确输出和错误输出都保存到同一个文件中
      命令 &> 文件 以覆盖的方式，把正确输出和错误输出都保存到同一个文件中
      命令 &>> 文件 以追加的方式，把正确输出和错误输出都保存到同一个文件中
      命令>>文件1 2>>文件2 以追加的方式，把正确输出输出到文件1，错误输出都输出到文件2中

    3. 输入重定向

      wc [选项] [文件名] 统计输入的字节数、单词数、行数

      ​ -c 统计字节数

      ​ -w 统计单词数

      ​ -l 统计行数

      命令 < 文件名 统计文件中的 字节数、单词数、行数

  2. 多命令顺序执行与管道符

    1. 多命令顺序执行

      符号 格式 作用
      ; 命令1;命令2 无逻辑联系
      && 命令1&&命令2 当命令1执行正确，才会执行命令2，当命令1执行不正确，命令2则不会执行
      || 命令1||命令2 当命令1执行不正确，命令2执行，命令1执行正确 ，不会执行 命令2

      举个栗子：

      dd if=输入文件 of=输出文件 bs=字节数 count =个数

      ​ 选项： if 指定源文件或源设备

      ​ of 制定目标文件或目标设备

      ​ bs=字节数 指定一次输入/输出多少字节，即把多少 字节看作一个数据块

      ​ count 制定输入/输出多少个数据块

    2. 管道符

      命令1 | 命令2 命令1的正确输出作为命令2的操作对象

      more 使文件的内容分屏显示

      ll -a /etc/ | more

      netstat -an 查询所有的网络连接

      grep [选项] “搜索内容” 文件名 在文件中搜包含”搜索内容“的内容

      ​ -i 忽略 大小写

      ​ -n 输出行号

      ​ -v 反向查找

      ​ –color=auto 搜索出的关键词用颜色表示

      netstat -an | grep “ESTABLISHED”

  3. 通配符与其他特殊符号

    1. 通配符

      通配符 作用
      ？ 匹配任意一个字符
      * 匹配0个或任意多个字符
      [] 匹配[]中任意一个字符
      [-] 例如：[a-z]匹配a-z中任意一个字符
      [^] 例如：[^123]匹配任意一个123以外的字符

    2. Bash中其他特殊符号

      符号 作用
      ‘ ‘ 单引号，在单引号中符号都没有特殊含义
      “” 双引号，在双引号中符号都没有特殊含义，但是”$“ 、”`”和”\“是例外，拥有”调用变量的值”、”引用变量”、”转义字符”的特殊含义
      `` 反引号，其中是系统命令
      $() 和反引号作用一样，推荐使用$()，易于区分
      # #开头表示注释
      $ 用于调用变量的值
      \ 转义符
